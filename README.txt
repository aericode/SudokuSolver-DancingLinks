Para compilar, acesse a pasta raiz via terminal e digite "make"

Para executar, digite "./run"

O menu irá abrir e irá propor resolver um entre os puzzles do arquivo

----------------------
Para adicionar puzzles, adicione uma linha extra em sudoku.csv, dentro de input

cada sudoku deve ter 16 caracteres, os caracteres vazios devem ser representados como um hifem (-)
--12-1-3----4-2-

Coloque uma vírgula e a resposta do outro lado.
